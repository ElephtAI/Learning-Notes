  • Dimensionality Reduction Techniques

    • Feature Selection (Keep the most relevant features)

      • Random Forest for Feature Selection: <BGI - No.6.1>
      • Missing Value Ratio (缺失值比率): remove the feature if its missing value ratio is higher than the preset threshold
      • Low Variance Filter (低方差滤波): remove the features with lowest variance (need Normalization at first sine variance depends on the range)
      • High Correlation Filter: remove one of feature pair, if they have high correlation compare to the thresold
      • Backward Feature Elimination: delete the feature with lowest effect on the model performance at each round
      • Forward Feature Selection: choose the feature that can mostly improve the model performance at each round

      • what is Eigenvalue: Eigenvectors are used for understanding linear transformations. In data analysis, we usually calculate the eigenvectors for a correlation or covariance matrix. Eigenvectors are the directions along which a particular linear transformation acts by flipping, compressing or stretching. Eigenvalue can be referred to as the strength of the transformation in the direction of eigenvector or the factor by which the compression occurs.
